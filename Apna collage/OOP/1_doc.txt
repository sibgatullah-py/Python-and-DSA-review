# Constructor
all classes have a function called __init__(), which is always executed when an object of that class is being initiated

class Car:
    def __init__(self,name,brand,color):
        self.name = name
        self.brand = brand
        self.color = color

car1 = car(name,brand,color)



IF we don't create a Constructor, python will automatically create a Constructor when the class is inherited  


## STATIC METHODS (decorator)
Methods that do not use self parameter (works at class level)
Class level methods are called static methods

*Decorators allows us to wrap another function in oder to extend the behavior of the wrapped function, without permanently modifying it.

Decorators wraps around an method changes it's behavior and returns us back 



# del keyword 
Used to delete object properties or object itself.

del s1
del s1.name
