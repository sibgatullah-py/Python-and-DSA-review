# Constructor

## STATIC METHODS (decorator)
Methods that do not use self parameter (works at class level)
Class level methods are called static methods

*Decorators allows us to wrap another function in oder to extend the behavior of the wrapped function, without permanently modifying it.

Decorators wraps around an method changes it's behavior and returns us back
